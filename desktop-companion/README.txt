披星云桌面伴侣 v3.2 - 启动说明
================================

快速启动
--------
1. 双击 "披星云伴侣-双击启动.bat" 或直接双击 "pixingyun-mate.exe"
2. 程序启动后：
   - 系统托盘（右下角）出现伴侣图标
   - 主界面窗口自动打开
3. 首次使用需登录披星云账号

系统托盘
--------
程序在后台运行，通过系统托盘图标管理：
  双击图标 — 打开主界面
  右键图标 — 菜单（打开主界面/快采/全量/暂停/退出）

图标颜色：
  绿色 = 运行正常
  橙色 = 采集中
  红色 = 有异常
  灰色 = 未登录/离线

功能模块
--------
仪表盘     — 运行状态总览
账号绑定   — 扫码绑定抖音/视频号/快手/小红书
数据采集   — 定时采集管理、Cookie状态、采集记录
抖店管理   — 抖店账号管理和数据同步
设置       — API地址、自动采集等配置
关于       — 版本信息和更新检查

支持的平台
----------
抖音 | 小红书 | 快手 | 视频号 | 抖店

便携特性
--------
无需安装，解压即用，不写注册表
数据全部在本地文件夹，可U盘携带
删除文件夹即卸载

常见问题
--------
Q: 双击没反应？
A: 首次运行需加载依赖，等待10-30秒，查看系统托盘。

Q: 如何完全退出？
A: 右键托盘图标 → 退出。关闭主窗口不会退出程序。

Q: 主窗口关了怎么打开？
A: 双击系统托盘图标。

Q: 首次运行较慢？
A: 需要下载Chromium浏览器（约150MB），仅首次。

开发调试
--------
运行 start.bat 或 python companion_app.py --debug
--debug 参数会保留控制台窗口以便查看日志
