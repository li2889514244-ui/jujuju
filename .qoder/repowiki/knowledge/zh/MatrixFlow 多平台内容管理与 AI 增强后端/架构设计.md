- **核心框架**: 采用 NestJS 模块化架构，通过 `app.module.ts` 聚合认证、用户、团队、账号、内容、AI、平台集成等十余个功能模块。
- **数据持久层**: 使用 Prisma ORM 管理 PostgreSQL 数据库，通过 `schema.prisma` 定义复杂的多租户（Organization/Team）及社交业务模型（Account/Post/DailyStats）。
- **安全与拦截**: 全局部署 `JwtAuthGuard` 和 `RolesGuard` 实现 RBAC 权限控制，配合 `ThrottlerGuard` 进行接口限流；通过 `TransformInterceptor` 统一 API 响应格式。
- **异步与实时通信**: 集成 `@nestjs/schedule` 处理定时数据同步与内容发布任务；利用 `Socket.IO` (ScanBindGateway) 实现扫码绑定等实时交互场景。
- **外部集成**: 封装多平台（抖音、小红书等）采集器与 OAuth 流程，并接入 Anthropic AI SDK 提供内容生成与优化服务。
